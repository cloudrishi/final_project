# Final Project - oaqjp-final-project-emb-ai (AI-Based Emotion Detection)
# https://github.com/cloudrishi/oaqjp-final-project-emb-ai

## Project Description
A Flask-based web application that detects emotions from text using IBM Watson NLP EmotionPredict service.

## Features
- Emotion detection using IBM Watson NLP
- Error handling for invalid inputs
- Web interface for user interaction
- RESTful API using Flask
